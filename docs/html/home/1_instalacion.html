1. Instalacion

1.1. Pre-requisitos

Para seguir este documento es necesario tener instalados los siguientes programas:

  • Python 3.10

    Para verificar si está instalado, ejecutar desde una terminal:

    python --version

    La respuesta debería ser similar a Python 3.10.*

  • Git

    Para verificar si está instalado, ejecutar desde una terminal:

    git --version

    La respuesta debería ser similar a git version 2.38.*

  • Node.js

    Para verificar si está instalado, ejecutar desde una terminal:

    node --version

    La respuesta debería ser similar a v18.12.1

  • VS Code o algún IDE para usar los Jupyter Notebooks

  • Credenciales válidas de Bitbucket

    Para eso, se debe dar de alta en Bitbucket.

    Y luego solicitar los permisos al administrador de los repositorios.

1.2. Instalación

Los siguientes pasos instalan la aplicación completa:

  • Core del Simulador

  • Servidor de Backend

  • Servidor de Frontend

Al finalizar esta sección, la estructura de carpetas debería quedar así:

|--app-simulador-radar
| |--simulador_core
|    |--backend_api.py
|    |--más contenido
| |--dependencias_estaticas
|    |--más contenido
| |--frontend
|    |--más contenido

  1. Abrir una terminal en la carpeta que contendrá a app-simulador-radar.

2. Con la terminal en la carpeta deseada, ejecutar:

mkdir app-simulador-radar && cd app-simulador-radar
git clone git@bitbucket.org:radaresfacet/simulador_core.git@dev
git clone git@bitbucket.org:radaresfacet/dependencias_estaticas.git
git clone git@bitbucket.org:radaresfacet/frontend.git

3. Usando Python 3.10.*, crear un nuevo ambiente. Se muestran las instrucciones para crear uno usando la herramienta venv. Para Windows:

cd simulador_core
python -m venv .venv
.venv\Scripts\activate
python -m pip install -U pip

4. Instalar los requerimientos del core y del backend (desde el ambiente):

pip install -r requirements.txt

5. Instalar los requerimientos del *frontend*.

npm install
