1. Instalación Rápida¶

Es necesario tener instalados los siguientes programas:

  1. Python 3.10

    Para verificar si está instalado, ejecutar desde una terminal:

    python --version

    La respuesta debería ser similar a Python 3.10.*

  2. Node.js

    Para verificar si está instalado, ejecutar desde una terminal:

    node --version

    La respuesta debería ser similar a v18.12.1

  3. Descargar la última versión del software desde este link. (Requiere permisos).

  4. Descomprimir el archivo

  5. Iniciar los servicios:

    Ejecutar el archivo app-simulador-radar/simulador_core/iniciar_servicios.bat.

    Ejecutar el archivo app-simulador-radar/simulador_core/linux_iniciar_servicios.sh.

  6. Consultar la seccion http://localhost:5000/static/html/home/2_uso.html.
